Octopress 固定ページの追加の仕方

octopressで新規の記事ではなく、固定のページをつくる方法

  • 自分のプロフィールであるaboutページを作ってみる
$ rake new_page['about']

生成された新しいページ用のファイルへの書き方は、記事を書くときと同じようにmarkdownで書く

$ vim source/about/index.markdown
  • ページのリンクを追加する

参照する場所がないとアクセスされないので、リンクを追加

$ vim source/_includes/custom/navigation.html

<ul class="main">
  <li><a href="{{ root_url }}/about">About</a></li>
  <li><a href="{{ root_url }}/">Blog</a></li>
  <li><a href="{{ root_url }}/blog/archives">Archives</a></li>
</ul>
