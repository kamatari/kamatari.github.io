静的サイトエンジン HUGOの使い方

インストール

今回はmac環境で、brewを使う。golangが必要なので、それも一緒にいれる。

$ brew install go
$ brew install hugo

もちろんgo getでも落とせるし、ここからhttps://github.com/spf13/hugo/releases ファイルをダウンロードでもOK

サイト作成

つくりたいサイトのpathを入れてhugo new site.

$ hugo new site /path/to/site

新規ページ、新規blog投稿

$ hugo new about-page.md
$ hugo new post/first_post.md

blogのテーマの取得

公式に置いてあるテーマはここから確認できる。

  • hugoThemes

スクリーンショットが無くて、何がどれやらという感じなので、最初は全部取ってきてもいいかもしれない。

$ git clone --recursive https://github.com/spf13/hugoThemes.git themes

ちなみに個別に取ってきたいなら、themesディレクトリを作って、その下にgit cloneして持ってくる。

$ mkdir themes
$ cd themes
$ git clone url-of-theme-which-you-want

blog をlocalで確認

さきほど丸ごと持ってきたthemeの中から好きな名前を指定。記事の中でdraft = trueになっている記事も表示するオプションをつけて、serverをたてる。

$ hugo server --theme=redlounge --buildDrafts

そうすると、コマンドのログにも出ているが、http://localhost:1313 から、構築したサイトの確認ができる。
