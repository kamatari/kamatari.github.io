Octopress + Github Pages で記事を投稿する方法

概要

octopressとGithub Pagesで構築したblogに記事を投稿する方法をまとめた

  • Octopress + Github Pagesでblogを構築した話

まず記事を書くファイルを生成する

$ rake new_post['title']

titleに入れた文字列は、記事のpermalinkのurlになる。例えばこの記事を作るときは以下のように1

$ rake new_post['how to post entry']

そうするとファイルが、sorce/_postsの下に生成される

記事の情報を編集する

生成したファイルを開くと以下の内容が記述してある

---
layout: post
title: "how to post entry"
date: 2014-01-19 14:30
comments: true
categories:
---

それぞれの項目についての説明

  • layout: 投稿のレイアウトのタイプを設定できる
  • title: 実際に記事のタイトルとして表示されるテキストをここに書く
  • date: ここに記載した時刻が記事の投稿時刻になる
  • comments: コメントを許可するかどうか
  • categories:カテゴリを設定可能

記事を書く

上記の設定の---の下からmarkdown形式で本文を書く それと、octopressにはmarkdown記法で書いた本文を投稿前にpreviewできる

$ rake preview

とすると、defaultでhttp://localhost:4000で、書いた記事が実際にどのように表示するか確認ができる

記事を投稿する

記事が書き終わって確認が出来たら、以下のコマンドでアップ

$ rake generate
$ rake deploy
  1. 必要に応じてrakeの前にbundle execをつける [return]
